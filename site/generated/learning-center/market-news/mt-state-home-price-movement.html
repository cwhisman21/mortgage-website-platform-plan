Montana home-price movement in the latest FHFA HPI

Quarterly, annual, and longer-run state HPI comparisons with a clear boundary between market movement and property value.

Maya Brooks Snap Mortgage Editorial Contributor Last updated July 10, 2026

Federal Housing Finance Agency

Key takeaways

  • FHFA purchase-only HPI: 723.7.
  • quarterly HPI change: -0.3%.
  • annual HPI change: 0.3%.
  • five-year HPI change: 44.9%.

The latest statewide index reading

Montana's latest FHFA purchase-only HPI is 723.7, accompanied by a quarterly change of -0.3%. For Montana, the 723.7 index level is a benchmark for repeat-sale price movement, not a dollar-denominated typical home price. A quarter in which the index fell by -0.3% can help a borrower date the statewide market context, but it cannot identify the value or asking price of a selected property. Reading 723.7 with -0.3% also keeps the level separate from the rate of change. Alongside the 723.7 state index, local inventory, property type, condition, concessions, taxes, insurance, and financing costs still need current evidence before the trend informs a decision.

Quarterly and annual windows tell different stories

Over the latest quarter, Montana's purchase-only HPI fell by -0.3%; over the latest year, it rose by 0.3%. The -0.3% figure is more sensitive to recent movement, while 0.3% compares the latest quarter with the matching quarter a year earlier. A borrower should not choose between the -0.3% and 0.3% windows based only on which looks more favorable, because the comparisons answer different timing questions. Together they show whether recent statewide movement is aligned with or different from the broader year without predicting the next quarter, a local bidding environment, or the price path of one home in Montana.

Place the latest movement in a longer sequence

Across the five-year comparison, the Montana HPI rose by 44.9%, compared with the latest annual movement of 0.3%. The 44.9% change places the current 723.7 index reading in a longer repeat-sale sequence, but it can span very different rate, inventory, and economic conditions. The 44.9% change should not be applied to an old purchase price to estimate a current property value. For a borrower, the contrast between 44.9% and 0.3% is most useful as a reminder that search assumptions can age at different speeds. Even when the 44.9% direction is clear, current comparable properties and a property-specific payment worksheet remain necessary.

Use the index to test local assumptions

The combination of a -0.3% quarterly change, 0.3% annual change, and 44.9% five-year change gives Montana borrowers three dated views of the same statewide index. If the -0.3%, 0.3%, and 44.9% windows point in different directions, the divergence shows that timing matters; if they point together, statewide consistency still does not erase local variation. A Montana search plan can use those three changes to ask whether current listings, seller concessions, and property types resemble the broad state movement. It should not use 723.7 as a sale price or use any of the percentage changes as a forecast, because FHFA HPI measures repeat-sale movement rather than the complete set of homes available to one borrower.

The HPI cannot value a property or set a payment

An HPI level of 723.7 does not establish a Montana property's dollar value, and an 0.3% annual movement does not establish how much any home gained or lost. The 723.7 index excludes the borrower's down payment, loan amount, interest rate, taxes, insurance, association costs, repairs, and time horizon. FHFA can also revise published values, so the -0.3% and 44.9% comparisons should retain their release period. The 723.7 index level must remain separate from the dollar figures used in a property review. Those limits make the 723.7 state index useful for orientation but unsuitable as an appraisal, offer recommendation, equity calculation, promised appreciation claim, payment quote, or underwriting conclusion.

Connect the state trend to current property evidence

A practical Montana borrower review can record -0.3%, 0.3%, and 44.9% as statewide trend markers, then compare them with current local listings and recent property-level evidence. For a purchase, keep the -0.3% state change as context while adding the selected price, inspection findings, appraisal process, taxes, insurance, cash-to-close, and loan disclosures. For a refinance or equity question, begin with a defensible property review rather than multiplying a prior value by 0.3% or 44.9%. This approach preserves the information in the 723.7 HPI while giving current local and borrower-specific facts the decision-making role that a statewide repeat-sale index cannot fill.

Methodology

This article uses the official FHFA purchase-only state quarterly HPI and calculates changes from matching prior-quarter, prior-year, and five-year index observations when available. Figures are reported as published for each cited geography and period. Derived percentages and differences use only the displayed source values; unavailable figures are omitted rather than estimated.

What this data cannot determine

FHFA may revise index values, statewide movement can mask local and property-type differences, and an index level is not a dollar-denominated home value. The evidence is broad market context, not a property valuation, rate or payment quote, eligibility finding, underwriting decision, product recommendation, or prediction.
