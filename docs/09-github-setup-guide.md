# GitHub Setup Guide

This local project folder is ready to connect to GitHub once a remote repo exists.

## Local Folder

```txt
C:\Users\caleb\OneDrive\Documents\GitHub\mortgage-website-platform-plan
```

## Simple GitHub Website Steps

1. Go to GitHub in your browser.
2. Click New repository.
3. Repository name: `mortgage-website-platform-plan`.
4. Visibility: private is recommended until the plan is cleaned up.
5. Do not add a README, `.gitignore`, or license on GitHub, because this local folder already has files.
6. Create the repository.
7. Copy the repository URL.
8. Send the URL back to Codex.

After that, Codex can connect this local folder to the GitHub repo and push the first commit.

## If GitHub CLI Is Installed Later

The repo can also be created from the terminal with GitHub CLI, but it is not currently installed on this machine.

## Current Limitation

Git is installed locally. GitHub CLI is not installed. Publishing to GitHub needs either:

- a GitHub remote URL from the browser-created repo, or
- installing/signing into GitHub CLI.
